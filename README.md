### HOW TO LAUNCH THE PROJECT

### The base URL is: "https://api.thecatapi.com/v1"

### 1st way is to Direct File Opening
   - Simply double-click on index.html

### 2nd way is to run the server using python
   - python -m http.server 8000
   - Then open "http://localhost:8000"

### Features

### 1. Breed Selector
   - Browse and select from a comprehensive list of cat breeds
   - Automatically fetch the breed's details when selected
   - View random cats or breed-specific images

### 2. Cat Image Gallery
   - Display random cat images based on your selections
   - Filter by breed or media type (all images, static images only, or animated GIFs)
   - View current cat information with breed details

### 3. Breed Information Panel
   - Breed Name - The name of the selected cat breed
   - Temperament - The personality traits of the breed
   - Life Span - How long the breed typically lives
   - Origin - The country or region the breed originated from

### 4. Voting System
   - Upvote - Mark a cat image as good
   - Downvote - Mark a cat image as not preferred
   - Track voting statistics in real-time
   - Each vote triggers a new cat image to appear

### 5. Favorites Gallery
   - Save your favorite cat images to a persistent gallery
   - Click thumbnails to view images in fullscreen mode
   - Refresh gallery to sync with The Cat API
   - Display gallery items in a responsive grid layout

### 6. Media Type Filter
   - All - Show all types of cat media
   - Static Images - Show only JPG and PNG files
   - Animated GIFs - Show only animated GIF files

### 7. Fullscreen Viewer
   - Click any favorite thumbnail to view it in fullscreen
   - Click anywhere to close the fullscreen view
   - High-quality image viewing experience

### How to Use the App

### 1. View a Random Cat
   - Click the "Next Cat" button to load a random cat image
   - The cat's information will appear in the "Breed Facts" panel

### 2. Filter by Breed
   - Select a breed from the "Breed Selector" dropdown
   - Click "Next Cat" to load an image of that specific breed
   - Breed details will update automatically

### 3. Filter by Media Type
   - Choose between:
   - All - Any cat images
   - Static Images - Photos only (JPG, PNG)
   - Animated GIFs - Moving images
   - Click "Next Cat" to apply the filter

### 4. Add to Favorites
   - When you find a cat you like, click the "Favorite" button
   - The image will be saved to your Favorites Gallery
   - A new cat will automatically load

### 5. Vote on Cats
   - Click "Upvote" if you like the current cat
   - Click "Downvote" if you don't
   - Your vote counts will update in the "Voting Stats" panel
   - A new cat will automatically load after voting

### 6. View Favorites
   - Scroll down to the "Favorites Gallery" section
   - Click any thumbnail to view it in fullscreen
   - Click the "Refresh Gallery" button to sync favorites

### 7. View Full-Size Image
   - Click any thumbnail in the Favorites Gallery
   - The image opens in fullscreen mode
   - Click anywhere on the screen to close
